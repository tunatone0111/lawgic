# Lawgic: 유사 판례 검색 서비스

## git commit message convention
### message format
[type] title
### type
- [feat]: 새로운 기능을 추가
- [fix]: 버그 수정
- [refactor]: 코드 리팩토링
- [style]: 코드 형식 변경(동작에 영향을 주는 코드 변경 없음)
- [test]: 테스트 코드에 대한 모든 변경사항
- [docs]: 문서 수정
- [chore]: 기타 사항
### example
- [refactor] web crawler refactor
- [feat] server code add
